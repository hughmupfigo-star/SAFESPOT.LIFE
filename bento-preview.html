What you're seeing. Left: the nav bar in its default state — the old "MENU" text is now a compact 3×3 bento icon (inline SVG so it never flashes). Right: tap it and the dark overlay fades in with a 3-column grid of square tiles. Eight page tiles — Home, Mission, The Hub, News, Shop, Publishing, About, FAQ — each with its own custom icon. The tile for the page you're on is highlighted white, so there's always a visual anchor. Below the grid, the Sign Petition CTA takes the full width as a horizontal primary tile. On phones under 480px the grid collapses to 2 columns.
