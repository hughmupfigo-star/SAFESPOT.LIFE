Accessibility Statement

Our Commitment

SAFE is committed to ensuring that its website is accessible to all users, including those with disabilities. We strive to provide an inclusive online experience for everyone.

Accessibility Features

This website includes the following accessibility features:

  • Semantic HTML markup for proper document structure
  • Sufficient colour contrast ratios for readability
  • Keyboard navigation support throughout the site
  • ARIA labels and descriptions for interactive elements
  • Responsive design that works on mobile devices
  • Text alternatives for images and visual content
  • Readable fonts and adjustable text sizing
  • Clear navigation and link descriptions

Browser & Assistive Technology Support

This website is designed to work with modern web browsers and assistive technologies including screen readers, keyboard-only navigation, and speech recognition software.

Known Limitations

We are continuously working to improve accessibility. If you encounter any accessibility barriers or have suggestions for improvement, please contact us.

Video & Audio Content

The music player on this site includes standard HTML5 audio controls. For optimal experience with audio content, use a modern browser with native HTML5 audio support.

Feedback & Reports

If you encounter any accessibility issues while using this website, please let us know so we can work to address them. You can report accessibility concerns to: enquiries@safespot.life

Standards Compliance

This website strives to comply with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards and the Americans with Disabilities Act (ADA) where applicable.

Contacting Accessibility Support

For accessibility-related inquiries or assistance, please contact us at enquiries@safespot.life with "Accessibility" in the subject line.
